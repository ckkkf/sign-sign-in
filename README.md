# sign sign in

#### 介绍
校友邦自动签到脚本（支持自定义经纬度签到） 一个基于 Python 3 的开源自动签到脚本，专为校友邦类实习/签到平台设计。支持自定义经纬度（模拟定位）提交签到，便于完成需要位置参数的签到场景。适合用于学习、测试与个人自动化，但请遵守目标站点服务条款与法律规定。
